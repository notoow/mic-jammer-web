External control box

아래 구성은 비간섭형 외부 제어 콘솔이나 상태 표시 박스를 바로 붙이기 위한 준비 항목입니다. 제어, 표시, 확인음 데모에만 초점을 둡니다.

  • 마이크로컨트롤러 1개: USB 시리얼 또는 BLE 연결용
  • 대형 순간식 버튼 1개: 현재 페이지와 같은 물리 토글 입력용
  • 상태 LED 1세트: safe, armed 상태 표시용
  • 소형 버저 또는 스피커 1개: 확인음 출력용
  • 전원 모듈 1개: USB-C 또는 5V 입력 정리용
  • 하우징 1개: 금속 또는 견고한 ABS 패널
  • 배선 및 브레드보드 또는 프로토보드: 빠른 프로토타입 조립용
